RONAALP
-------

Reduced Order Nonlinear Approximation with Active Learning Procedure [1],[2]

Source code available `here
<https://github.com/cscherding/RONAALP>`_.

Installation
------------
Follow these commands to install RONAALP::

   git clone https://github.com/cscherding/RONAALP.git
   cd RONAALP
   python setup.py install
   
Examples
--------
A demonstration jupyter notebook can be found in the `examples <https://github.com/cscherding/RONAALP/examples>`_ folder. Data for training and testing can be requested from the author upon reasonable request.
